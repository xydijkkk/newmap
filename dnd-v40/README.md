# DND v4.0 map set

This directory contains 23 original, unlabeled map backgrounds generated for the
dual-world SillyTavern character card project:

- 2 overview maps
- 12 Faerun regional maps
- 9 Nine Hells layer maps

All production images are uncompressed 1536 x 1024 PNG files. Location labels,
markers, and navigation controls are rendered by the character card at runtime.
These are original compositions and are not scans or reproductions of published
official maps.
