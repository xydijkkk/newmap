# DND Character Creator Portraits v1

Original AI-generated character portraits for the SillyTavern DND character creator.

- `originals/`: lossless per-species PNG crops used by the selected-species preview.
- `thumbs/`: lightweight JPEG thumbnails used by the 18-species selection grid.
- `source-sheets/`: the three generated 3x2 source sheets.

The portraits depict original characters. They do not contain Baldur's Gate 3 characters, logos, interface artwork, or copied costumes. Product names mentioned in the surrounding project remain the property of their respective owners.
